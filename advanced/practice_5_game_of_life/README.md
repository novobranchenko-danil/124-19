# Game of Life - Cellular Automaton Simulator
## Описание:

Симулятор клеточного автомата "Игра Жизнь" с визуализацией эволюции поколений в PNG и GIF форматах.

## Установка:

pip install -r requirements.txt

## Использование:
```
python gol.py [аргументы]

Аргументы командной строки:

--input - Входной CSV файл (по умолчанию: init.csv)

--generations - Количество поколений (по умолчанию: 10)

--cell-size - Размер клетки в пикселях (по умолчанию: 100)

--debug - Включить режим отладки (по умолчанию: False)
```
## Примеры команд
### Запуск с параметрами по умолчанию
python gol.py

### Свои настройки
python gol.py --input data.csv --generations 50 --cell-size 150

### Режим отладки с тестами
python gol.py --debug

### Формат входного файла
```
Файл CSV со следующей структурой:

13          # количество строк
13          # количество столбцов
1,3         # координаты живой клетки (строка 1, столбец 3)
1,4         # нумерация с 1
...
```
## Выходные данные
### Программа создает:

generation.csv - история всех поколений в CSV формате

000_init_grid.png - начальное состояние

001_generation.png, 002_generation.png, ... - каждое поколение

evolution.gif - анимация всех поколений

## Файловая структура проекта
```
├── unit test file/     # тестовые файлы
├── gol.py              # основной код
├── config.py           # конфигурация и аргументы командной строки
├── requirements.txt    # зависимости (Pillow)
├── init.csv            # пример входного файла
└── README.md           # эта документация
```

## Правила "Игры Жизнь"
Живая клетка с <2 соседей умирает (одиночество)

Живая клетка с 2-3 соседей выживает

Живая клетка с >3 соседей умирает (перенаселение)

Мертвая клетка с 3 соседами оживает

## Требования
Установить зависимости из requirements.txt:
pip install -r requirements.txt

## Примечания
При отсутствии указанного файла используется init.csv

Старые PNG/GIF/CSV файлы удаляются при каждом запуске

В режиме --debug выполняются unit-тесты
