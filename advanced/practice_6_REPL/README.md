# ZIP Code Lookup System

Простая консольная утилита для работы с почтовыми индексами США.

## Функционал

- **loc** — Поиск информации по ZIP-коду (город, штат, координаты)
- **zip** — Поиск ZIP-кодов по названию города и штата
- **dist** — Расчет расстояния между двумя ZIP-кодами в милях
- **end** — Завершение программы
- **return** — Возврат в главное меню

## Структура проекта
```
├── main.py                 # Основной цикл программы
├── console_handler.py      # Вывод информации и запросов
├── data_handler.py         # Логика обработки данных
├── utils.py                # Вспомогательные функции
├── zip_util.py             # Чтение данных из CSV
├── zip_codes_states.csv    # База данных ZIP-кодов
└── README.md               # Эта документация
```

## Формат данных

CSV-файл `zip_codes_states.csv` содержит следующие колонки:
- ZIP код (5 цифр)
- Широта (decimal degrees)
- Долгота (decimal degrees)
- Город
- Штат (2 буквы)
- Округ

Каждая строка преобразуется в список:  
`['12180', 42.673701, -73.608792, 'Troy', 'NY', 'Rensselaer']`

## Особенности

- Валидация ввода: ZIP-код — 5 цифр, город/штат — только буквы
- Координаты выводятся в формате DMS (градусы-минуты-секунды)
- Расстояние рассчитывается по формуле Хаверсаина
- Обработка специальных команд в любом поле ввода
