# Farmers Market Base

Консольное приложение для поиска и просмотра информации о фермерских рынках США с возможностью оставлять отзывы, оценивать рынки и управлять данными (для администраторов).

## Функционал

- **login/register** — Авторизация и регистрация пользователей (пароли хешируются с bcrypt)
- **markets** — Просмотр всех рынков с пагинацией и средним рейтингом
- **search** — Поиск рынков по различным критериям:
  - По FMID (точное совпадение)
  - По названию (частичное совпадение)
  - По городу и штату (точное совпадение штата + частичное совпадение города)
  - По ZIP-коду (точное совпадение)
- **nearby** — Поиск рынков в радиусе 30 миль от указанного:
  - От конкретного рынка по FMID
  - От первого рынка, найденного по названию
  - От первого рынка в указанном городе и штате
  - От рынка в указанном ZIP-коде
- **help** — Справка по командам
- **q** — Выход из программы

### Дополнительные возможности:
- Просмотр детальной информации о рынке (контакты, соцсети, принимаемые виды оплаты, сезоны работы)
- Оставление отзывов с оценкой от 1 до 5
- Просмотр всех отзывов о рынке (сортировка по рейтингу)
- Автоматический расчет среднего рейтинга
- Удаление отзывов — пользователи могут удалять только свои отзывы
- Удаление рынков — функция доступна только для администраторов

## Формат данных

### Export.csv (источник: USDA Farmers Markets Directory)
Содержит информацию о фермерских рынках США:
- FMID — уникальный идентификатор рынка
- MarketName — название рынка
- Website, Facebook, Twitter, Youtube — ссылки на соцсети
- street, city, County, State, zip — адрес
- Season1Date, Season1Time — сезоны и время работы
- x, y — географические координаты (долгота, широта)
- Credit, WIC, SNAP — принимаемые виды оплаты
- Organic, Bakedgoods, Vegetables и др. — типы продуктов

Каждая строка преобразуется в список с индексами из `MarketIndex`.

**Тестовая запись:** В файле `Export.csv` добавлена специальная запись с FMID `9999999` и названием "Test Market For Delete" для тестирования функции удаления рынков.

### reviews.csv
Хранит отзывы пользователей:
- FMID, MarketName, zip — информация о рынке
- comment — текст отзыва
- rating — оценка (1-5)
- user — имя пользователя
- date — дата и время отзыва

### users.json
Хранит зарегистрированных пользователей:
```json
{
    "username": "$2b$12$...хеш...",
    ...
}
```
Пароли хранятся в виде bcrypt-хешей.

### permission.json
Хранит права доступа пользователей:
```json
{
    "admin": "delete"
}
```
Пользователи с правом "delete" могут удалять рынки из базы данных.

### .env (тестовые учетные данные)
Для удобства тестирования в проекте предусмотрен файл `.env` с реальными учетными данными администратора:
- **Логин:** admin
- **Пароль:** admin

Эти данные позволяют протестировать функцию удаления рынков без необходимости создавать нового администратора.

## Установка и запуск

```

python main.py
```

## Управление данными

### Удаление отзывов
При просмотре отзывов о рынке доступна команда `[d] delete`. 
- Удалить можно только те отзывы, которые были написаны текущим пользователем
- При попытке удалить чужой отзыв выводится сообщение: "You can only delete your own reviews"
- После подтверждения отзыв удаляется как из текущего списка, так и из файла `reviews.csv`

### Удаление рынков (только для администраторов)
При просмотре детальной информации о рынке доступна команда `[d] delete`, но только для пользователей с правами администратора.

**Тестирование функции удаления:**
1. Войдите в систему под учетной записью администратора:
   - Логин: `admin`
   - Пароль: `admin` (указаны в файле `.env`)
2. Найдите тестовый рынок с FMID `9999999` (специально добавлен в `Export.csv` для тестирования)
3. В детальном просмотре рынка введите `d` для удаления
4. Подтвердите удаление

**Важно:** При удалении рынка:
- Запись удаляется из текущего списка результатов
- Запись удаляется из файла `Export.csv` (необратимо)
- Все отзывы, связанные с этим рынком, остаются в `reviews.csv` (связь по FMID)

## Особенности

- Валидация ввода: ZIP-код — 5 цифр, FMID — только цифры, оценка — от 1 до 5
- Хеширование паролей с помощью bcrypt
- Постраничный вывод результатов поиска
- Расчет расстояния между рынками по формуле Хаверсаина
- Поддержка русского языка в отзывах (UTF-8)
- Система прав доступа для административных функций
- Удаление только собственных отзывов
- Тестовые учетные данные для проверки функционала администратора
